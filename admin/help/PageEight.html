    Get Seller Lots
    Back To The Top
    >>> Seller User Name: Get Seller Lots
    This function lets you as the admin retrieve all of the lots that belong to one seller by entering their "username" into the field provided and clicking on the "Get seller lots" page.
    Get A Lot
    Back To The Top
    >>> Lot Id : Get Lot
    This function lets you, the admin to retrieve a lot, but entering the lot number, or numbers seperated by a comma (1,2,3). To do this, enter a lot number(s) in the Lot ID field and then click on the "Get lot" button. The resulting page will retrieve the lots that you have typed in and present information about each lot.
    GET A USER
    Back To The Top
    This function lets you, the admin to retrieve a user by a number of methods that you can select via radio buttons below. You retrieve a user by either:

    >> Username
    >> E-Mail Address
    >> First Name
    >> Last Name
    >> Organization

    The resulting page will bring back a list of matches.

    The columns in the table are explained here below:

    COLUMN 1 (Marked as "?")
    This column contains a checkbox that allows you to select users you would like to delete from the database. Once you have selected all the users from the displayed list by checking the checkmark box in this colmun you can click on the "Delete Users" button at the bottom of the page to delete those selected. N.B. Deleting a user also deletes all bids made by that user.

    COLUMN 2 (Marked as "userID (edit) )
    This column simply tells you the userID that the user has been assigned. It is a clickable link that takes you to a page where you can edit the user information for the particular user that you click on.

    COLUMN 3 (Marked as "Status")
    This column simply give you a "Status" link that when clicked on takes you to a STATUS page that has four radio button options for status:

    () Not yet approved
    () Approved
    () On Probation
    () Banished

    - At the top of the table found on this page, the User Verify Code is also displayed. As the admin on this page you can change the status of the user by clicking on any of the radio buttons and then clicking on the "Update Status" button below.

    COLUMN 4 (Marked as "Bids")
    This is the Bids column that contains a hyperlink to a page that displays the bids that the user has made in your auction website.

    >>> BIDS PAGE:
    This page allows you to view the bids for the listed bidder that you have chosen to view bids for.

    Displayed are two tables. The first of which has a sequential list of bids that the user has made along with information on the bid. The first column of this table marked as "Edit Bid" displays the lot number and can be clicked on to edit the bid of the user.

    The second table displayed is a summary of bids. It is divided into four organized categories:

    >> Summary of all Bids (all bids)
    >> Winning Bid Summary (all the bids that the user has won)
    >> Winning Bids Summary (all the bids that are currently winning in open lots for the user)
    >> Winning Bids Summary (all the winning bids on closed lots for the user)

    COLUMN 4 (Marked as "Username")
    This column simply displays the username

    COLUMN 5 (Marked as "Name (Last,First) )
    This column simply displays the last and first name of the user.

    Search For Lots, Criteria Based
    Back To The Top
    This section allows you as the admin to select certain search criteria for either bids or lots and display them any way you like. N.B. Choosing to view lots will list all options for that lot. Choosing bids will list the bids for those lots instead.

    ROW 1:
    In this row you can define what type of lots you would like to view and how you would like the resulting query to be sorted. You can choose to apply any of the following criteria to the lots on this row:

    [] Inactive
    [] Preview
    [] Open
    [] Closed

    You can also choose how you would like the results displayed.

    Order By:
    [Lot Opening Time]
    [Lot Closing Time]
    [Lot Name]
    [Lot id]
    [Number of Bids]

    These can be chosen to be sorted in either ascending or descending order.

    ROW 2:
    In this row you as the admin can choose to view bids along with criteria for the bids you want to choose.

    ()Winning Bids Only ()All bids

    You can also choose how you would like the results displayed.

    Order By:
    [Bidder Name]
    [Bidder Username]
    [Bid date/time]
    [current bid]
    [initial bid]
    [maximum bid]
    [bid quantity]
    [quantity won]

    These can be chosen to be sorted in either ascending or descending order.

    ROW 3: LOT OPENS:
    This section allows you to define a time period for the lot opening that will relate to other search criteria that you have specified in this section to enter your query search.

    ROW 4: LOT CLOSES:
    This section allows you to define a time period for the lot closing that will relate to other search criteria that you have specified in this section to enter your query search. ROW 5: CATEGORY:
    This row simply allows you to apply the above search criteria to one, multiple, or all categories. To select more than one category but not all, hold the [CTRL] key down while clicking with your mouse on the caegories you would like the search query to apply to.

    To submit your query, click on the "Submit Lot Query" button.
